# Customer Behavior Analysis

## 📌 Project Overview

Customer Behavior Analysis is a data analytics project that explores customer purchasing patterns, spending habits, and product preferences using SQL, Python, and Power BI. The project aims to transform raw customer transaction data into actionable business insights that can support data-driven decision-making.

Through data cleaning, analysis, and visualization, this project identifies key customer trends, purchasing behavior, and business opportunities to improve customer satisfaction and revenue growth.

## 🎯 Objectives
* Analyze customer purchasing behavior and spending patterns.
* Identify high-value customers and customer segments.
* Discover popular product categories and sales trends.
* Evaluate customer purchase frequency and loyalty.
* Generate business insights through interactive dashboards.
  
## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL
* Power BI
* Jupyter Notebook
* VS Code
## 📂 Dataset

The dataset contains customer shopping and transaction records, including customer demographics, product purchases, spending amounts, ratings, payment methods, discounts, and purchase frequency. These attributes help analyze customer behavior, purchasing trends, and overall business performance.
